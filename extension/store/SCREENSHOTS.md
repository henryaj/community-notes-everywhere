# Chrome Web Store Screenshots

Descriptions of screenshots to capture for the store listing. Recommended size: 1280x800 or 640x400.

## Screenshot 1 — Highlighted notes on a page

Show a news article or blog post with 2-3 inline highlights visible (green for helpful, amber for pending). Demonstrates the core experience of browsing a page that has community notes.

## Screenshot 2 — Note popover with ratings

Click on a highlighted note to show the popover with the note body, author info, status badge, and rating pills (Yes / Somewhat / No). Ideally show a note that has several ratings to demonstrate the consensus mechanism.

## Screenshot 3 — Creating a new note

Select text on a page to show the "Add Note" button, then capture the note creation form with the textarea and source self-check radio buttons. Shows how easy it is to contribute.

## Screenshot 4 — Extension popup with user profile

Show the popup panel with the user's avatar, handle, reputation score, karma, and note/rating counts. Demonstrates the reputation system and user dashboard.

## Screenshot 5 — Note visibility toggle

Show the toggle between "Helpful only" and "All notes" modes, with amber/grey highlights appearing when "All notes" is selected. Demonstrates the filtering capability.
